# BF0009

Input parameter is provided but the Brainfuck source does not contain the input command `,`.

## Severity

Hidden — the code compiles and runs normally. The parameter is simply ignored.

## Trigger

This diagnostic is reported when the method has a `string`, `System.IO.Pipelines.PipeReader`, or `System.IO.TextReader` parameter but the source contains no `,` command.

## Example

```cs
using Esolang.Brainfuck;

partial class Sample
{
    [GenerateBrainfuckMethod("+")]
    public static partial void NoInputNeeded(string input);
}
```

## Fix

Either remove the unused input parameter, or add `,` to the Brainfuck source if input was intended.
